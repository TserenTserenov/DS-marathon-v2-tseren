- **Примечание:** авторство методологии — отдельный вопрос, но в рабочей тетради участника автором указан «Управляющий сети Александр»

## 3. Технические проблемы PDF

### 3.1. Размер файла
- **p1_merged.pdf:** 23.5 МБ (46 стр., A4)
- **Рабочая_тетрадь_участника.pdf:** 145 КБ (40 стр., letter)
- **Проблема:** файл в ~160 раз тяжелее. Вероятно, каждая страница содержит растровый фон или изображения-маски. Это критично для отправки в Telegram и email.

### 3.2. Отсутствие навигационных закладок (outline)
- **p1_merged.pdf:** нет outline/bookmarks
- **Рабочая_тетрадь_участника.pdf:** нет outline (ограничение LaTeX по умолчанию)
- **Рекомендация:** добавить закладки по разделам (Введение, ЭТАП 0–3, ВСТРЕЧА 1–4, Приложение) — это критично для 46-страничного документа

### 3.3. Поиск по файлу
- **Текстовый слой:** присутствует (pdftotext извлекает текст)
- **Но:** если фон растровый, поиск может работать медленно и некорректно на некоторых ридерах
- **Рекомендация:** убедиться, что шрифты встроены (embedded), а текст — векторный, не растр

### 3.4. Размер страницы
- **p1_merged:** A4 (595×842 pts)
- **Рабочая_тетрадь_участника:** Letter (612×792 pts)
- **Рекомендация:** для цифровой дистрибуции лучше A4 или оптимизированный letter. Текущий A4 — OK.

## 4. Рекомендация по подходу

### Вариант А: Правки в текущий файл Инги (если это InDesign/Word/другой исходник)
Если у Инги есть редактируемый исходник — передать ей **список правок из разделов 1–2** этого документа. Плюс попросить:
1. Встроить шрифты и убрать растровые фоны → уменьшить вес до < 2 МБ
2. Добавить навигационные закладки (PDF bookmarks)
3. Сделать гиперссылки кликабельными

### Вариант Б: Взять оформление Инги за основу и пересобрать самим
Если файл Инги — это `Рабочая_тетрадь_участника.pdf` (LaTeX via pandoc), то:
1. У него **отсутствуют emoji** и современные ссылки
2. У него **нет закладок** и нет тегов доступности
3. **Но:** вес 145 КБ — идеален, текст чистый, поиск работает отлично

**Оптимальный путь:**
- Взять `workbook_combined.md` (актуальный текст)
- Сгенерировать PDF через pandoc + шаблон оформления, близкий к дизайну Инги
- Добавить hyperref-закладки (`--pdf-engine=xelatex` + `toc-depth: 2`)
- Встроить emoji через `emoji` LaTeX-пакет или заменить на иконки
- Размер выйдет ~200–500 КБ вместо 23 МБ
