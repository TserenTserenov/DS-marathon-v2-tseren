# Программа марафона «Первые шаги в IWE»

Весь материал марафона выдаётся через **Telegram-бота [@aist_me_bot](https://t.me/aist_me_bot)**.

Бот присылает задания каждый день, помогая выстроить ритм работы на протяжении двух недель.

---

## Что такое IWE (простыми словами)

**IWE** — это ваш интеллектуальный экзоскелет. Как костюм Iron Man усиливает Тони Старка, IWE усиливает вас в вашей работе.

- Не заменяет ваше мышление — **усиливает** его
- Помнит ваши проекты и контекст — не начинает с чистого листа
- Растёт вместе с вами — чем больше вы работаете, тем умнее он становится
- Помогает работать по специальности быстрее и системнее

**Марафон** — это 14-дневный эксперимент: пробуете IWE на деле, не бросая свою работу.

---

## Как это работает

1. **Запустите бота** → [@aist_me_bot](https://t.me/aist_me_bot)
2. **Получайте задания ежедневно** — утром теория, днём практика
3. **Выполняйте задания** — бот отслеживает прогресс
4. **Встречайтесь с наставником** — после недели 1 и недели 2

## Неделя 1: Где я сейчас? → Первые практики

| День | Теория 📘 | Практика 🔧 |
|:----:|-----------|-------------|
| **1** | Три состояния застревания — Хаос, Тупик, Поворот. Какое твоё? | **Диагностика** — Напиши текст "Моё текущее состояние" (0,5 стр) |
| **2** | Экзоскелет vs протез — почему IWE усиливает, а не заменяет | **Первый слот** — Выдели 15–25 мин на работу по специальности сегодня |
| **3** | Почему крутимся как белка в колесе — реактивность vs стратегия | **Трекер-помидоро** — Заполни трекер за сегодня |
| **4** | Информационный шум и потеря фокуса — внимание как ресурс | **Мимолётные заметки** — Запиши 3+ мысли по ходу дня |
| **5** | Цикл выгорания — почему рывки не работают | **Стоп-моменты** — Отследи 3+ момента ухода внимания |
| **6** | Почему советы не работают — лайфхаки vs система | **Гипотеза** — Сформулируй: "Если я буду..., то..." |
| **7** | Итоги недели 1 — рефлексия и анализ | **Чек-лист недели** — Что получилось? Что сложно? |

### Результат недели 1
- Определено текущее состояние (Хаос / Тупик / Поворот)
- Запущены первые практики
- Сформулирована гипотеза на неделю 2

| День | Теория 📘 | Практика 🔧 |

| **14** | Стиль жизни созидателя — ежедневные практики | **План на месяц** — Что продолжишь делать после марафона? |

### Результат недели 2
- Освоены базовые практики
- Понимание своих мемов и стоп-моментов
- Готовый план развития после марафона

## Расшифровка практик

| Практика | Описание |
|----------|----------|
| **Диагностика** | Письменное описание текущего состояния жизни |
| **Слот** | Фиксированное время для работы по специальности (15–25 минут) |
| **Трекер** | Ежедневная фиксация выполнения практик |
| **Мимолётные заметки** | Быстрые записи мыслей и идей по ходу дня |
| **Стоп-моменты** | Отслеживание моментов, когда внимание уходит |
| **Гипотеза** | Формулировка эксперимента: "Если я буду X, то Y" |
| **Рабочий продукт (РП)** | Конкретный результат работы (текст, схема, план) |
| **Мемы** | Убеждения, влияющие на поведение |

## Связь с наставником

Бот — это ежедневная поддержка между встречами с наставником:

```
Бот (дни 1–7)  →  Встреча с наставником  →  Бот (дни 8–14)  →  Финальная встреча
```

Наставник помогает:
- Интерпретировать результаты диагностики
- Скорректировать практики под вас
- Разобрать сложности и стоп-моменты
- Закрепить результаты марафона
