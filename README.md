# Марафон систематичности 🚀

Программа личного развития для тех, кто хочет **выйти из хаоса** и начать **системно двигаться вперёд**.

Шаг за шагом выстраивая осознанный стиль **Жизни Созидателя**. Без героизма и мотивационных качелей.

---

## 🎯 Сущность марафона

**Это не курс и не интенсив**, это начало нового, более осознанного стиля жизни.

Марафон состоит из **4 встреч** и помогает:

| Этап | Результат |
|------|-----------|
| **Встреча 1** | Определить текущее состояние (Хаос / Тупик / Поворот) |
| **Встреча 2** | Выстроить минимальный ритм саморазвития (15-25 мин в день) |
| **Встреча 3** | Перейти от размышлений к рабочим продуктам |
| **Встреча 4** | Заложить основу устойчивого развития |

```
Опросник → Встреча 1 → Встреча 2 → Встреча 3 → Встреча 4
   ↓           ↓           ↓           ↓           ↓
Состояние   Фокус       Метод       Анализ     Система
```
